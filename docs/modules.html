getBranches

  • getBranches<T>(guard: (o: unknown) => o is T, columns: [string, Parser?][], repositoryPath: string, gitOptions: string[] | BranchListOptions, pattern?: string, columnDelimiter?: string): Promise<T[]>

  • Returns information about Git branches in application-defined JavaScript objects. The JavaScript objects are created from the return of the GitAnnexAPI.branch method.

    Type parameters

    • T

    Parameters

    • guard: (o: unknown) => o is T

      The type predicate to validate each branch.

        • (o: unknown): o is T

        • Parameters

          • o: unknown

          Returns o is T

    • columns: [string, Parser?][]

      Maps the columns returned by the --format option to property names. A parser to convert the string to another data type may be provided for each column.

    • repositoryPath: string

      The path of the repository to interrogate.

    • gitOptions: string[] | BranchListOptions

      The BranchListOptions for the command.

    • Optional pattern: string

      Filters branches using either fnmatch(3) or matching completely or from the beginning up to a slash.

    • Optional columnDelimiter: string

      The column delimiter used by the --format option. If unspecified, \t is used to split at %09 in --format.

    Returns Promise<T[]>

    An array containing objects passing the type guard function.

getLineStarting

  • getLineStarting(str: string, prefix: string, includePrefix: boolean): string | undefined

  • Searches a multi-line string and returns the first line begining with the specified prefix. The prefix may be included with or omitted from the return string.

    Parameters

    • str: string

      The string to be searched.

    • prefix: string

      The string to locate at the beginning of a line.

    • includePrefix: boolean

      A flag indicating whether the prefix is to be included in the return value.

    Returns string | undefined

    The requested line or undefined if the prefix was not located.

getLineStartingAsArray

  • getLineStartingAsArray(str: string, prefix: string): string[]

  • Searches a multi-line string and returns the first line begining with the specified prefix as a string array. The return array is created by removing the prefix from the beginning-of-line and splitting the rest of the line at each space character.

    Parameters

    • str: string

      The string to be searched.

    • prefix: string

      The string to locate at the beginning of a line.

    Returns string[]

    The requested line as a string array. The array is empty if the prefix was not located.
